# 📊 株価分析ダッシュボード

日本の株価データを取得・分析し、可視化するWebアプリケーションです。
StreamlitとPlotlyを使用しています。

## 機能
- **銘柄検索**: 証券コードを入力してデータを取得
- **業種別分析**: JPXのデータに基づく業種ごとのスクリーニング
- **株価チャート**: 過去1年間の株価推移と移動平均線（25日/75日）
- **出来高チャート**: 日々の売買代金の可視化
- **財務指標**: PER, PBR, ROEなどによる絞り込み

## 使用技術
- Python 3.11
- Streamlit
- Pandas
- yfinance
- Plotly
